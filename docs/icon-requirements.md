# App Icon Requirements

## Google Play Store

Required app icon:

- 512 x 512 PNG
- 32-bit PNG
- Transparent background allowed, but clean solid-background icons usually look better

Recommended additional Android-ready assets:

- Adaptive icon foreground
- Adaptive icon monochrome

## iOS App Store

Recommended source icon:

- 1024 x 1024 PNG

From this source, generate all required iOS sizes during app asset preparation.

## Suggested USCIS Tracker Icon Design

Simple concept:

- dark navy or deep blue background
- white or light-blue document/checklist icon
- small tracking pin, magnifier, or shield accent
- avoid tiny text inside the icon

Good visual directions:

- document + check mark
- shield + receipt line
- folder + status badge

## Design Guidelines

- keep the icon simple and recognizable at small size
- avoid detailed text or long labels
- use strong contrast
- keep spacing balanced with generous padding
- make sure the icon still looks good when cropped into circles or rounded shapes
- test it on a real phone home screen
