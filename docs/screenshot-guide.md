# Play Store Screenshot Guide

Recommended size:

- 1080 x 1920

Google Play minimum:

- at least 2 phone screenshots

Recommended set:

## 1. Home Screen Search

What to show:

- receipt number field
- nickname field
- premium dark header
- language selector

Caption:

- Track USCIS cases in a clean, simple dashboard

## 2. Case Status Result

What to show:

- loaded case result
- status badge
- progress bar
- timeline steps

Caption:

- See your current status and progress at a glance

## 3. My Cases Screen

What to show:

- multiple saved cases
- last updated times
- refresh and delete actions visible

Caption:

- Save multiple cases and manage them in one place

## 4. AI Support Screen

What to show:

- chat conversation
- user and AI bubbles
- “Powered by Groq” text in the support header

Caption:

- Ask AI support for simple USCIS guidance anytime

## 5. Language Support

What to show:

- UI switched to Spanish or Hindi
- same premium layout

Caption:

- Use the app in English, Spanish, or Hindi

## Screenshot Tips

- use realistic but safe sample receipt numbers
- avoid blank or broken states
- keep screenshots bright, sharp, and readable
- make sure no debug text is visible
- capture on a real device when possible
- if any placeholder text remains, replace it before taking screenshots
